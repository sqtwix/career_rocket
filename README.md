# Career Rocket
**Career Rocket** — веб-приложение для помощи трудоустройства и анализа рынка IT-вакансий. Проект разработан на Python с использованием фреймворка Bottle. Позволяет выбрать тариф менторства, просматривать вакансии со сторонних сайтов, просматривать статистику по вакансиям, зарплатам и категориям в IT-сфере.

---
## 🎯 Описание проекта

Career Rocket помогает IT-специалистам и соискателям по нахождению работы и ориентированию на рынке труда. Приложение предоставлеяет тарифы помощи по трудойстройству, собирает и визуализирует данные по вакансиям, зарплатам, количеству соискателей и работодателей в различных IT-категориях.

**Ключевые возможности:**
- Различные предложения помощи по трудойстройству
- Просмотр выбранных вакансий на сторонних сайтах 
- Интерактивные диаграммы распределения вакансий
- Детальная информация по зарплатам (мин., ср., макс.)
- Статистика по категориям (Web, ML, Mobile, DevOps и др.)
- Современный адаптивный интерфейс

## 🛠 Технологии

### Backend
- **Python 13** — основной язык разработки
- **Bottle** — микрофреймворк для веб-приложений

### Frontend
- **HTML5**, **CSS3**
- **Bootstrap 3** — для адаптивной вёрстки
- **Chart.js** — для построения графиков
- **Font Awesome** — для иконок

### Форматы данных
- **JSON** — хранение статистики (имитация БД)

  
## 🚀 Установка и запуск

### Предварительные требования
- Python 3.9 или выше
- Git (опционально)

### Инструкция по запуску

1. **Клонировать репозиторий**
```bash
  git clone https://github.com/yourusername/carrer-rocket.git
  cd career-rocket
```

2. **Создать виртуальное окружение (рекомендуется)**
```bash

# Для Windows
python -m venv venv
venv\Scripts\activate

# Для Linux/Mac
python3 -m venv venv
source venv/bin/activate
```

3. **Установить зависимости**:
```bash
pip install bottle
# или если есть файл requirements.txt
pip install -r requirements.txt
```

4. **Запустить сервер**
```bash
python app.py
```

5. **Открыть в браузере**
Перейдите по адресу: http://localhost:5555
